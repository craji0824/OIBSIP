## 📂 Repository Structure

The repository is organized into three distinct project directories:

### 1.  [Landing Page](https://landing-page-nine-omega-62.vercel.app)
A clean, premium, and fully responsive landing page showcasing product features, dynamic hover states, responsive layouts, and interactive elements.
- **Tech Stack**: HTML5, CSS3

### 2.  [Personal Portfolio](https://landing-page-nine-omega-62.vercel.app)
A professional, modern portfolio website showcasing skills, projects, background, and certifications.
- **Features**: 
  - Smooth-scrolling sections
  - Elegant light/dark mode theme toggle
  - Responsive layout (mobile & desktop)
  - Interactive projects grid and contact form
- **Tech Stack**: HTML5, CSS3, Vanilla JavaScript

### 3.  [Temperature Converter](https://tempconverter-six.vercel.app)
A high-fidelity, premium glassmorphism temperature converter card with modern design aesthetics.
- **Features**:
  - Simultaneous Celsius, Fahrenheit, and Kelvin conversions
  - Temperature-dependent dynamic background gradient themes (Cold, Normal, Hot, Very Hot)
  - Temperature status badges (🥶 Freezing, 😊 Cool, ☀ Pleasant, 🔥 Very Hot)
  - Persistent Recent History panel (stores last 5 conversions in local storage)
  - Custom animated mouse cursor with hover state scaling
  - Keyboard execution on `Enter`
  - Fully responsive and accessibility-checked
- **Tech Stack**: HTML5, CSS3, Vanilla JavaScript

##  Getting Started

All projects are client-side only and can be run locally without any compiler or server configuration:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/craji0824/OIBSIP.git
   cd OIBSIP
   ```

2. **Run a project**:
   Navigate into any directory and open `index.html` directly in your browser, or start a local HTTP server:
   ```bash
   # From any project folder
   python -m http.server 8080
   ```
   Open `http://localhost:8080` in your web browser.
