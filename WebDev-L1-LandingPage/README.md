# NexaAI - AI Landing Page (Oasis Infobyte Task 1)

Hey there! 👋 This is my submission for **Task 1 (Level 1)** of the **Oasis Infobyte Web Development Internship**. 

I built a landing page for a fictional product called **NexaAI**, which is a productivity app powered by AI. My main goal was to make it look super modern, sleek, and fully responsive using just raw HTML, CSS, and basic JavaScript (no big frameworks!).

---

## 🎨 What I Built (Features)

*   **Modern Dark Theme**: Designed a premium-looking dark mode using clean HSL colors, glowing gradient backdrops, and a subtle grain overlay.
*   **Interactive Workspace Preview**: The hero section has a simulated chat window showing NexaAI typing out a response in real-time.
*   **Bento Grid Layout**: Used CSS Grid to create a responsive bento grid for the features section. It looks great on desktop and automatically stacks on mobile!
*   **How it Works Timeline**: A simple step-by-step guide showing how users can connect their apps, let the AI learn, and start automating their work.
*   **Pricing Cards**: 3 different plans (Starter, Pro, Enterprise) with a highlighted "Popular" tag for the Pro version.
*   **Wall of Love (Testimonials)**: Added card layouts with nice gradients to display user feedback.
*   **Scroll Animations**: Used the browser's native JavaScript `IntersectionObserver` to trigger fade-in animations as you scroll down.
*   **Mobile-Friendly**: The navbar collapses into a working hamburger menu on smaller screens.

---

## 🛠️ Tools Used

*   **HTML5** for page structure (using proper tags like `<nav>`, `<header>`, `<section>`, etc.)
*   **Pure CSS3** for layouts (Flexbox & CSS Grid), animations, and custom variables
*   **Vanilla JavaScript** for the mobile menu toggle and scroll animations
*   **Google Fonts** (Inter and Outfit)
*   **Lucide Icons** for the modern vector icons
